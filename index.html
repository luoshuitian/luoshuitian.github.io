1. 取消全局配置

如果想在一台电脑上使用多个 GitHub 账号，使用全局配置是不行的。如果使用全局配置的话，那么每次使用 git 命令时都会使用这个全局的用户名和邮箱，就没有办法区分不同的 GitHub 账号了。
